WAR! Web Map — GitHub Pages Prototype

1) Create a GitHub repo.
2) Put this 'docs' folder in the repo root.
3) In GitHub: Settings → Pages → 'Deploy from a branch' → Branch: main → Folder: /docs

Open the Pages URL. The map saves GM edits locally in your browser (localStorage).
Exports:
- Territories_Status.csv
- Adjacency.csv

Offline/no-JS map:
- Use your existing No-JS HTML file in iPhone Files preview.
